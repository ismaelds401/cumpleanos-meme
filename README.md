# Feliz Cumpleanos Amor

Juego romantico hecho con Angular para celebrar los 23 anos de una enamorada. Incluye preguntas, rompecabezas, personalizacion de la cumpleanera, camino de recuerdos, 6 llaves, cofre con musica y un portal para formar a la cumpleanera con sus cualidades.

## Comandos

```bash
npm install
npm start
```

Luego abre `http://localhost:4200/`.

Tambien puedes usar:

```bash
npm run build
ng serve
```

## Personalizacion

- Cambia preguntas y explicaciones en `src/app/data/questions.data.ts`.
- Cambia imagenes, respuestas del camino, look correcto y cualidades del portal en `src/app/data/game-config.ts`.
- Coloca tus fotos en `public/assets/img/` y usa rutas como `assets/img/tu-foto.png`.
- Cambia colores globales en `src/styles.css`.
